# Particles

Ce répertoire contiendra les textures et les configurations de particules :
- Fichiers PNG ou atlas pour les confettis, étincelles, fumée, etc.
- Fichiers JSON (ou `.atlas`) décrivant les animations de particules que vous chargerez avec `this.add.particles`.

Les particules seront utilisées pour les effets de gain, les jackpots et les animations de spin.
