# Audio Assets

Ce répertoire contiendra les effets sonores du jeu :
- **Spin** (rouleaux qui tournent)
- **Stop** (arrêt du rouleau)
- **Win / Jackpot** (son de gain)
- **Lever / Bouton** (interactions UI)
- Bruit de pièces, de monnaie, etc.

Les fichiers seront chargés avec `this.load.audio` dans `BootScene`.
