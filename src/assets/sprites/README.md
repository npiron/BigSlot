# Sprites

Ce répertoire contiendra tous les sprite‑sheets du jeu :
- Cadre complet de la machine à sous
- Bande de rouleaux (reel strip)
- Symboles individuels (Bar, 7, fruits, etc.)
- Icônes UI (bouton spin, pièces, jackpot)

Les fichiers seront importés dans `BootScene` avec `this.load.spritesheet`.
