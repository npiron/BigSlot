# Backgrounds

Ce répertoire contiendra les images de fond du jeu :
- Fond de salle de casino (1920 × 1080) pour le décor principal.
- Textures de panneaux, néons, rideaux, etc.
- Images de transition (fade‑in/out) si besoin.

Les fonds seront chargés avec `this.load.image` ou `this.load.backgroundColor` dans `BootScene`.
